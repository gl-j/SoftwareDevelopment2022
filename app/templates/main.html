This is the main page of the project Loan Predictability

Features:

  This page will ask the user a set of questions, which will determine whether the user will get a loan or not based on a Logistic Regression model, that has been previously trained with existing data.

Requierements:

  Python 3.9.x, Flask 1.1.x, Numpy 1.21.x, Pandas 1.3.x, Scikit Learn 0.24.x, Pickle 4.0

  Seaborn 0.11.x, Matplotlib 3.4.x
