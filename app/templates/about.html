  Data set taken from: https://www.kaggle.com/altruistdelhite04/loan-prediction-problem-dataset The data set is already presented in the train-test form.

About the data:

  The data is from the company "Dream Housing Finance" which provides home loans. In order for a customer to apply for a loan, first the company validates the customer eligibility for loan. Company wants to automate the loan eligibility process based on customer detail provided while filling online application form. These details are Gender, Marital Status, Education, Number of Dependents, Income, Loan Amount, Credit History and others. To automate this process, they have given a problem to identify the customers segments, those are eligible for loan amount so that they can specifically target these customers.

  This is a classification problem where it must be predicted if a loan would be approved or not
